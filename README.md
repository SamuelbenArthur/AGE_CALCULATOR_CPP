# AGE CALCULATION CPP

## Project Overview

This project is a C++ program that calculates a person's age based on their birth date and the current date. The program determines the age in years, months, and days.

## Objectives

* Accept a user's birth date.
* Accept the current date.
* Calculate the user's age.
* Display the age in years, months, and days.

## Features

* User-friendly input prompts.
* Calculates age accurately using the entered dates.
* Displays results clearly.
* Beginner-friendly C++ implementation.

## Technologies Used

* C++
* Dev-C++
* Standard Library (iostream)

## How to Run

1. Open the project in Dev-C++.
2. Compile the program.
3. Run the program.
4. Enter the birth date.
5. Enter the current date.
6. View the calculated age.

## Sample Input

Birth Date:

* Day: 15
* Month: 8
* Year: 2002

Current Date:

* Day: 10
* Month: 6
* Year: 2026

## Sample Output

Age:
23 Years, 9 Months, 26 Days
