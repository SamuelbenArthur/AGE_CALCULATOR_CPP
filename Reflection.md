# Reflection

## 1. What problem did I solve?

I developed a C++ program that calculates a person's age using their birth date and the current date.

## 2. What C++ concepts did I practise?

* Variables
* Data types
* User input (cin)
* Output (cout)
* Arithmetic operations
* Conditional statements (if)
* Arrays
* Program logic

## 3. What errors did I face?

I encountered compilation errors caused by missing brackets, incorrect variable names, and logical errors during the age calculation process.

## 4. How did I correct the errors?

I carefully reviewed the compiler messages, corrected the syntax mistakes, tested the program with different dates, and verified that the calculations were correct.

## 5. What did I learn from the engineering stretch activity?

I learned that programming can automate calculations that engineers perform regularly. It also improved my logical thinking and problem-solving skills.

## 6. What will I improve next time?

Next time, I will add leap year support, input validation for invalid dates, and automatically retrieve the current system date instead of asking the user to enter it.
